#  Quiz Competition – Python CLI App

A lightweight terminal-based quiz game built with Python. It features multiple-choice questions from geography, history, science, and current events, with real-time scoring and feedback.

---

## 🚀 Features
- Multiple-choice interface  
- Real-time scoring  
- Personalized feedback  
- No external libraries

---

## 🛠 Tech Stack

| Component      | Details                      |
|----------------|------------------------------|
| Language       | Python 3.12+                 |
| Interface      | Command-Line (CLI)           |
| Execution      | `python quiz.py`             |
